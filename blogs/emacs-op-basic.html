emacs 基本操作指令整理

<Control>-x <Control>-f <Alt>-x find-file 查找文件并在新缓冲区打开
<Control>-x <Control>-v <Alt>-x find-alternate-file 查找另外一个文件替换掉当前已读入的文件
<Control>-x i <Alt>-x insert-file 在当前光标位置插入文件
<Control>-x <Control>-s <Alt>-x save-buffer 保存文件
<Control>-x <Control>-w <Alt>-x write-file 保存文件为…
<Control>-x <Control>-c <Alt>-x save-buffers-kill-emacss 保存并退出 emacs
<Control>-h <Alt>-x help 帮助
<Control>-h t <Alt>-x help-with-turorial 启动教程
