1 unetbootin安装时报错: unetbootin menu.c32 not a com32r image

  • T430/PC安装正常，T440安装时报com32r image
  • syslinux版本不一致导致，从GitHub Unetbootin下载最新的生成
  • Another answer: hit "TAB"; the installer will show a list of possible images type the correct image name (watch caps!); hit enter. See https://askubuntu.com/questions/85254/cannot-boot-from-usb-disk-with-not-a-com32r-image-error
  • https://superuser.com/posts/367935/revisions
  • https://bbs.archlinux.org/viewtopic.php?id=171629

2 debian 9.3.0 packagekitd holds the /var/lib/dpkg/lock

  • apt-get install git causes open error(11)
  • because the service unattended-upgrades running: service --status-all
  • discuss: https://unix.stackexchange.com/questions/71746/can-i-disable-unattended-debian-package-updates
  • https://wiki.debian.org/PackageKit
  • https://wiki.debian.org/UnattendedUpgrades

3 error: No package 'gtk+-3.0' found

install gtk-3-dev: sudo apt-get install gtk-3-dev

4 error: xwidgets requested but WebKitGTK+ not found.

sudo apt-get install libwebkitgtk-3.0-dev libwebkitgtk-dev libwebkit2gtk-4.0-dev

5 emacs 26.0.90 support xwidget

config like as below: ./configure --with-cairo --with-xwidgets --with-x-toolkit=gtk3 ./configure --with-xwidgets --with-x-toolkit=gtk3
